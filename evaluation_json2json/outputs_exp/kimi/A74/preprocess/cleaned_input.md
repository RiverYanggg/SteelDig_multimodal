# Effect of Al Content on Inclusions in Fe-15Mn-xAl-C Low-Density Steel

Lei Cao, Guocheng Wang,\* Yuanyou Xiao, and Ke Tao

Three experiments of Fe-15Mn-xAl-C (x = 1, 3, 5) low-density steel with different Al contents are carried out, and the characteristics of inclusions are observed through field-emission scanning electron microscopy. The results show that the types of inclusions in the three samples are almost the same, mainly $\mathsf { A l } _ { 2 } \mathsf { O } _ { 3 } , \mathsf { A l N } ,$ MnS, TiN, ${ \sf A l } _ { 2 } { \sf O } _ { 3 }$ -MnS, AlN-MnS, TiN-MnS, and ${ \sf A l } _ { 2 } { \sf O } _ { 3 }$ -AlN-MnS. The number percentage of ${ \sf A l } _ { 2 } { \sf O } _ { 3 }$ decreases, while the number percentage of AlN increases with the increase of Al content in the three samples. As the Al content increases, the number of inclusions between 1 and 5 μm decreases, while the number between 5 and 10 μm increases. FactSage8.2 software calculation shows that an increase in Al content leads to an increase in the liquidus and a decrease in the solidus of the steel; at the same time, it causes the austenite region to narrow and expands the ferrite region. The change in the solidification route leads to a delay in the solidification fraction of AlN, MnS, and TiN inclusions precipitation. At last, the 2D mismatch theory shows that the mismatch degrees of $\mathbf { \Delta A l _ { 2 } O _ { 3 } } / \mathbf { M n S } ,$ , AlN/MnS, TiN/MnS, and $\mathsf { A l N } / \mathsf { A l } _ { 2 } \mathsf { O } _ { 3 }$ are 8.41%, 8.94%, 11.65%, and 11.66%, respectively.

# 1. Introduction

In recent years, Fe–Mn–Al–C low-density steel has been applied in important fields such as automobiles, ships, and aerospace, due to its excellent wear resistance, corrosion resistance, and lightweight.[1–5] Mn element has the effect of expanding the austenite zone, and its induced twinning effect is beneficial for hindering the propagation of cracks in the material, significantly improving the plasticity of the steel.[6] When the Al content in the steel increases by 1%, the density of the steel decreases by about 1.3%.[7,8] The high content of Al can reduce the strain hardening rate of the steel, and improve the formability and low-temperature toughness.[9] However, in the smelting process L. Currently, some work has been conducted on the impact of changes in Al content on the characteristics of inclusions in medium/high Mn steel. Yu et al.[10] found that the increase in Al content resulted in an increase in the types of inclusions, and the morphology evolution order of inclusions is: spherical irregular (polyhedral) spherical. Alba et al.[11] found that the content of Al increased from 0.5% to 3%, and the number percentage of AlN-MnS inclusions in Fe-5Mn-xAl-C steel sharply increased. Wang et al.[12] studied that the inclusion with the highest number of inclusions in steel changed from MnS to AlN, as the Al content increased from 0.87% to 6.76%. In addition, they found that AlN inclusions could

form in the liquid phase when the Al content is higher than 3.28%. Xin et al.[13] believed that the number density of inclusions in Fe-16Mn-xAl-0.6C steel increased first and then decreased, while the size of inclusions showed a change of first decreasing and then increasing when the Al content increased from 0.002% to 2.1%. Park et al.[14] investigated that the number percentage of AlN inclusions in Fe-20Mn-xAl steels gradually increased as the Al content increased from 1% to 6%. Wang et al.[15] investigated that when the Al content increased from 0.002% to 2.1%, the main type of inclusions in Fe-16Mn-xAl-C steel changed from MnO/(MnO MnS) to AlN. They also found that AlN inclusions can worsen the hot ductility of steel in samples with Al content of 0.033% and 0.54%. Wang et al.[16] studied that the main composite inclusions transform from ${ \mathrm { A l } } _ { 2 } { \mathrm { O } } _ { 3 }$ -MnS to AlN-MnS inclusions with the increase of Al content. At the same time, the number percentage of AlN-MnS inclusions first increases and then decreases. Meanwhile, Jin et al.[17] confirmed through experiments and theoretical calculations that AlN can provide heterogeneous nucleation sites for MnS inclusions. In our previous study, we reported that an increase in Mn content is beneficial for expanding the FCC zone and leading to an advance in the solidification fraction of AlN, MnS, and TiN inclusions precipitation.[18] However, there are few reports on the effects of changes in Al content on the solidification route and inclusion formation of Fe-15Mn-xAl-C steel.

In the current work, three smelting experiments of Fe-15MnxAl-C (x = 1, 3, 5) low-density steel with different Al contents were conducted under laboratory conditions to study the effect of Al content on the solidification route and inclusions formation of steel. The formation mechanism of various types of inclusions was discussed based on changes in Al content.

# 2. Experimental Section

# 2.1. Smelting Process

Three smelting experiments of Fe-15Mn-xAl-C (x = 1, 3, 5) lowdensity steel with different Al contents were conducted under laboratory conditions. The first smelting experiment process is as follows: First, iron powder (700 g) and ferrous oxalate $( \mathrm { F e C } _ { 2 } \mathrm { O } _ { 4 } { \cdot } \mathrm { H } _ { 2 } \mathrm { O }$ , 1.575 g) are placed in an alumina crucible and then placed in a tubular furnace for heating. When the tubular furnace was heated to 1873 K, kept for 40 min, and then deoxidizer was added. First, 225 g of ferromanganese was added and kept for 20 min. Then 11 g of aluminum particles were added that were wrapped in the pure iron sheet and kept for 60 min. Finally, the sample was cooled to room temperature in a tubular furnace. The operating steps and the amount of ferromanganese added in the three groups of experiments were consistent. The mass of pure aluminum particles added to the second and third groups was 28 and 48 g, respectively. # 2.2. Analytical Methods

The chemical composition of three steels was detected using chemical analysis methods. The content of T.O and T.N was measured by a carbon sulfur analyzer. The carbon sulfur analyzer detected the content of C and S. Inductively coupled plasma mass spectrometry measured the content of Al, Si, Ti, Mn, and P.

The inclusions in the experimental samples were observed in two methods. 1) Metallographic observation. Solid steel was cut into cylinders using an electric discharge wire-cutting machine. Subsequently, metallographic sandpaper and metallographic polishing paste were used to process the sample into a mirror surface. 2) Preparation of inclusion powder by nonaqueous solvent electrolytic separation. After electrolytic separation, a magnet was used to separate the inclusion particles from the solution containing ethanol and collect them on a carbon strip.[19–21] The samples prepared by the above two methods were used for field-emission scanning electron microscopy (Model: ZEISS ΣIGMA HD) with energy dispersive spectrometer (Model: Oxford-X-Max 50 mm2 ) of the morphology and type of inclusions. # 4.1. Effect of Al Content on the Solidification Route of Steel

To investigate the effect of Al content on the solidification route of three samples, the phase diagram module in FactSage 8.2 was used to calculate the Fe–C–Si–Mn–P–S–Al–O–N–Ti system and the calculation results are shown in . shows the Figure 6 Table 3phase composition of the corresponding numbers in Figure 6. It can be seen that the solidification route of 1 # sample is: $\mathrm { L } \longrightarrow \mathrm { L } + \mathrm { F C C } \longrightarrow \mathrm { F C C } _ { : }$ , while the solidification route of 2 # and 3 # samples is: $\mathrm { L \longrightarrow L + B C C \longrightarrow L + B C C + F C C \longrightarrow L + F C C \longrightarrow }$ FCC and $\operatorname { L } \to \operatorname { L } + \operatorname { B C C } \to \operatorname { L } + \operatorname { B C C } + \operatorname { F C C } \to \operatorname { B C C } + \operatorname { F C C } \to \operatorname { F C C } ,$ respectively. The increasing content of Al leads to the solidification route of 2 # and 3 # samples passing through the BCC phase, and the area where the FCC phase exists decreases.

The liquidus temperature and solidus temperature of the samples were obtained using the FactSage 8.2. The liquidus temperature (TL) of 1 # to 3 # samples are 1702, 1740, and 1796 K, and the solidus temperature $( T _ { \mathrm { { S } } } )$ are 1588, 1560, and 1532 K, respectively. The liquidus temperature increases with the increasing content of Al, while the solidus temperature decreases. The precipitation of inclusions is closely related to the liquidus temperature and solidus temperature. Therefore, it is necessary to discuss the effect of changes in Al content on the formation of various inclusions in the steel.

# 4.3. Precipitation Mechanism of Complex Inclusion

Multiple types of composite inclusions were observed in the detection of experimental results, namely, $\mathsf { A l } _ { 2 } \mathsf { O } _ { 3 } { \cdot } \mathsf { M } \mathsf { n } \mathsf { S } .$ , AlN-MnS, ${ \sf A l } _ { 2 } { \sf O } _ { 3 } { \cdot } { \sf A l N } { \cdot } { \sf M n S } ,$ and TiN-MnS. Through the observation results, it can be clearly observed that there is an interface relationship between the composite inclusions. The precipitation order of single-particle inclusions is: $\mathrm { A l _ { 2 } O _ { 3 } } > \mathrm { A l N } > \mathrm { M n S } > \mathrm { T i N } .$ . Therefore, it can be speculated that the later formed inclusions can nucleate and grow at the heterogeneous nucleation sites of the first formed inclusions. To investigate the effectiveness of MnS inclusions nucleation on $\mathrm { A l } _ { 2 } \mathrm { O } _ { 3 } ,$ AlN, TiN surfaces, and AlN inclusions nucleation on ${ \mathrm { A l } } _ { 2 } { \mathrm { O } } _ { 3 }$ surfaces, the mismatch degree (δ) as suggested by Bramfit,[47] substrates’ capacity to promote heterogeneous phase nucleation was assessed. The calculation of mismatch is shown in Equation (23):

$$
\delta_ {\left(\mathrm{hkl}\right) _ {\mathrm{n}}} ^ {(\mathrm{hkl}) _ {\mathrm{s}}} = \frac {1}{3} \left[ \sum_ {i = 1} ^ {3} \frac {\left| d _ {[ \mathrm{uvw} ] _ {\mathrm{s}}} ^ {i} \cos \theta - d _ {[ \mathrm{uvw} ] _ {\mathrm{n}}} ^ {i} \right|}{d _ {[ \mathrm{uvw} ] _ {\mathrm{n}}} ^ {i}} \right] \times 100 \% \tag{23}

# 5. Conclusion

By studying the effect of different Al contents on the formation of inclusions in Fe-15Mn-xAl-C (x = 1, 3, 5) steel, the main conclusions are as follows: 1) The types of inclusions in Fe-15Mn-xAl-C $( x = 1 , 3 , 5 )$ low-density steel are almost the same, mainly $\mathrm { A l } _ { 2 } \mathrm { O } _ { 3 } ,$ , AlN, MnS, TiN, $\mathbf { A l } _ { 2 } \mathbf { O } _ { 3 } { \cdot } \mathbf { M } \mathbf { n } \mathbf { S }$ , AlN-MnS, TiN-MnS, and $\mathrm { A l } _ { 2 } \mathrm { O } _ { 3 ^ { - } }$ AlN-MnS. As the Al content in the three samples increases, the number percentage of ${ \mathrm { A l } } _ { 2 } { \mathrm { O } } _ { 3 }$ decreases, while the number percentage of AlN increases. The number percentage of ${ \mathrm { A l } } _ { 2 } { \mathrm { O } } _ { 3 }$ -MnS inclusions increases with the increasing content of Al, while the number percentage of AlN-MnS and TiN-MnS inclusions decreases. As the Al content increases, the number of inclusions between 1 and 5 μm decreases, while the number between 5 and 10 μm increases; 2) The increasing content of Al leads to an increase in the liquidus and a decrease in the solidus of the steel. At the same time, it causes the FCC region to narrow and expands the BCC region, and the solidification route of steel samples with 3% and 5% Al content passes through the BCC region; 3) The changes in the liquidus, solidus, and solidification route of the sample result in changes in the precipitation time of inclusions. When the Al content increases from 1% to 3%, the solidification route changes from FCC to FCC and BCC, which leads to an advance in the solidification fraction of AlN precipitation. Furthermore, when the Al content increases from 3% to 5%, although the solidification route is still FCC and BCC, high content of Al leads to a significant decrease in N solubility. Therefore, the comprehensive results of Al and N elements delay the precipitation of AlN. However, the solidification fraction of MnS and TiN inclusions precipitated is delayed with the increasing content of Al; and 4) The mismatch degrees of $\mathsf { A l } _ { 2 } \mathsf { O } _ { 3 } / \mathsf { M n S }$ , AlN/MnS, TiN/MnS, and $\mathsf { A l N } / \mathsf { A l } _ { 2 } \mathsf { O } _ { 3 }$ are 8.41%, 8.94%, 11.65%, and 11.66%, respectively. This indicates that $\mathrm { A l } _ { 2 } \mathrm { O } _ { 3 } ,$ AlN, and TiN could provide nucleation sites for the nucleation of MnS, and ${ \mathrm { A l } } _ { 2 } { \mathrm { O } } _ { 3 }$ could also provide nucleation sites for AlN.
